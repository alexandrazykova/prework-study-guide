# Prework Study Guide Webpage

## Description

This Prework Study Guide was created for boot camp students who were going through the Prework. It contains organized notes on HTML, CSS, Git, and JavaScript for learning and reviewing purposes.

## Installation

N/A

## Usage

As each section contains notes about certain topic, it makes navigating easy and simple. By pressing Command+Option+I (macOS) OR Control+Shift+I (Windows) OR right click anywhere on the page and selecting 'inspect' to open the DevTools console and see the JavaScript running. It will list the topics we learned from prework, and choose one topic to study first.
